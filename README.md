# o-chat

API to manage Users.



### Backend tech
<ul>
  <li>Node.js 12</li>
  <li>Typescript</li>
  <li>Express.js</li>
  <li>Rest Api</li>
  <li>MongoDB</li>
  <li>Mongoose ODM</li>
  <li>Mocha & Chai for testing</li>
  <li>log-to-file for logs</li>
  <li>Eslint - Airbnb Style</li>
</ul>

Must include header - X-Api-Key
